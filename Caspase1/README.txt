The files in this folder are as follow:
1. Caspase_data_with_BindingDB_IDs.xlsx : All Caspase inhibitors with SMILES string and BindingDB IDs, molecular descriptors, MACCS descriptors, Morgan finger prints (MPF_), and solvation free energies.
2. Caspase_data.csv : File used for training and test set generations
3..Sample_notebook.ipynb : Python notebook sample of all machine learning codes 
4. rdkit_descriptors.py : Python script for RDkit descriptors and fingerprint generation from a parent 3D-SDF file
